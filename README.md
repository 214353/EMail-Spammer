# EMail-Spammer

Send any email any number of times — straight from the terminal. No external libraries, no APIs, just pure Python.
 
---
 
## Features
 
- ✅ Zero dependencies — uses Python's built-in `smtplib`
- ✅ Dynamic input — recipient, subject, body, and count at runtime
- ✅ Send confirmation before blasting
- ✅ Per-message success/fail feedback
- ✅ Final delivery summary
